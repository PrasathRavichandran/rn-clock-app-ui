## App contains features like,
- View current date and time 
- Add Alarm
- Setting Timer
- Stopwatch

## Technology used:
In this project I have used [React native - Expo](https://expo.dev/), [React navigation](https://reactnavigation.org/) for mobile app navigation, [Dayjs](https://day.js.org/) for parse, validate and manipulate dates and times, [React native gesture handler](https://docs.swmansion.com/react-native-gesture-handler/) and much more 游땒

## Development Server
Run `npm run start` to run the metro server in your device. Then, choose options from the terminal to run the project in your simulator or emulator.
